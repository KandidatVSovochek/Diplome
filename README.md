## Описание проекта
Продукт: GitHub
DitHub это хостинг для IT-проектов. Веб-платформа нужна для контроля изменений в коде, версий продукта, а также хранит предыдущие версии кодов проекта. Дает возможность разработчикам вместе работать над задачей.
**Ссылка на проект по ручному тестированию**
https://krutevaed.yonote.ru/share/8beb82d6-cd1f-4996-8807-80af6cf6ed82
**Запуск тестов для последующего формирования отчета**
В терминале ввести команду *pytest --alluredir allure-result*
Будет создана папка allure-result
**Формирование отчета**
В терминале ввести команду *allure serve allure_result*
**Просмотр результатов**
1. Когда отчет сформирован автоматически откроется браузер
2. Перейти в раздел **Suites**
3. Нажать на название теста. Слева на панеле отобразятся шаги выполнения
**Запуск тестов без формирования отчета**
В терминале ввести команду *pytest -m "UI"* или *pytest -m "API"* или *pytest*
